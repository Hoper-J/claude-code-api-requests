**Tooltip** — 轻量的悬停/聚焦浮层，给一个简短到容易看不懂的标签补上一行上下文（状态徽标、缩写 chip）。浅色卡片面 + pop 阴影；悬停与键盘聚焦时显现。

```jsx
<Tooltip tip="抓取这一版时 MCP 还没连上——是采集当时的状态，不是该版本移除了 MCP。">
  <Badge tone="accent">MCP 未连接</Badge>
</Tooltip>
```

- 包住触发元素；wrapper 是 `inline-flex`，紧贴子元素。
- 触发元素靠近视口顶部时用 `placement="bottom"`（避免浮层在上方被裁切）。
- `tip` 为空时只渲染子元素、不出浮层——传可能为空的字符串也安全。
- 当标签本身含义不直白时优先用它而非原生 `title`：可被发现、合品牌、键盘可达、触屏可见。
